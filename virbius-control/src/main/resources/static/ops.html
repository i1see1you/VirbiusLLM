新建规则默认为 draft（不进执行面）；启用（上线） 后变为 active 并参与匹配。停用后恢复为 draft 可再编辑。

编辑

Groovy L3 脚本返回 [action: '…']，取值 allow / block / captcha / review。 dry_run 或 canary 未进桶时命中应返回 review；full 命中返回 block（或 captcha）。 引擎按规则 enforce_mode / canary_percent 解析为对外 effective_action。
